## 小米优购



小米优购本项目使用原生微信小程序 + less 开发，项目分为四个主页面，分别是首页，分类页，购物车，个人中心页面，还有其他子页面，商品详情，商品列表，订单列表，地址列表，结算页面, 客服页面。
